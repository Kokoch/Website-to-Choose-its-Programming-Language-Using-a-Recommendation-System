BSP PROJECT S1:

Choose its Programming Language Using a Recommendation System

The idea of my project is to make a website where I can post some investigations about programming languages to help people to choose the “right” one.

Main Objectives:

The principal objective of my project is to help people to choose a programming language for their activity. To achieve this objective, it is therefore necessary to provide complete documentation about the different programming languages chosen, in order to best help the user in his choice. However, I will not do the research on these languages. Indeed, I will use the documentation of my classmates to enrich my website. But if my project can only be summarized as making my classmates research available to everyone on the Internet, it wouldn't be interesting. This is why I decided to integrate an algorithm into my website that would allow me to advise the user on his choice.

So, I will make my website a recommendation system.

Scientific Section

The scientific aspect of the project is the research for “How to make a recommendation system?”. First of all, I will do some research to find websites that advise the user in his choice of programming language. And see if they use an algorithm or if they are just articles that only present some languages. In addition, I will do research on how a recommendation system works and how to create it. But also research databases, how they work, how to link them to an algorithm so that you can use your data to advise a user. In a second step, I will document myself on Html, Css, JavaScript and the Bootstrap library to be able to have my site created.

Technical Section

It’s about making a functional website coded in HTML, CSS and JavaScript to which several internet pages will be linked together. The objective of the project is to have an algorithm on the website that would allow users' data to be entered, and then used to process the data using a recommendation system and finally return an appropriate programming language.
